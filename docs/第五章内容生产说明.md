# 第五章内容生产说明

## 交付目标

第五章“搜索与回溯”已从空章节补成可学习、可练习、可判题、可复盘的内容闭环。本轮优先完成教学内容、OJ 和视频前期脚本，Manim 场景、预览图和成片留给下一轮生产。

## 课时与练习矩阵

| 编号 | 课时 | 核心模型 | 配套练习 |
| --- | --- | --- | --- |
| C05-01 | 枚举与搜索树 | `dfs(pos)`、决策层、叶子 | 二进制串枚举、各层节点计数 |
| C05-02 | DFS 基本框架 | 出口、候选循环、下一层 | 定长序列、叶子数 |
| C05-03 | 选择、递归、撤销 | `push -> dfs -> pop` | 固定 1 的二进制串、合法括号 |
| C05-04 | 全排列 | `used` 数组、字典序 | 全排列、第 k 个排列 |
| C05-05 | 组合枚举 | `start`、剩余名额 | k 数组合、组合求和 |
| C05-06 | 子集枚举 | 选/不选二叉分支 | 子集输出、子集和计数 |
| C05-07 | 迷宫 DFS | 方向数组、边界、`visited` | 可达性、简单路径数 |
| C05-08 | 剪枝 | 可行性剪枝、上界剪枝 | 剪枝子集和、回溯背包 |
| C05-09 | N 皇后简化版 | 列与两类对角线约束 | 方案数、第一个方案 |
| C05-10 | 回溯复杂度直觉 | 分支因子、深度、节点上界 | 搜索预算、完整树节点数 |

## 已完成的内容层

- `frontend/src/data/curriculum.ts`：10 个课时的标题、导语、核心概念、理解路径、完整 C++ 示例和 20 道练习入口。
- `backend/app/content/search_problems.py`：20 道题的题面、输入输出、数据范围、公开样例和隐藏测试。
- `frontend/src/data/support.ts`：6 题章节测验、6 项掌握清单、3 组回刷路径，以及 20 道题的三层提示、题解要点和常见错误。
- `docs/视频脚本/C05-01-*.md` 至 `C05-10-*.md`：课时定位、分镜、完整旁白、代码画面映射和练习衔接。

## 视频生产约定

1. 场景统一放在 `manim/scenes/search_backtracking/`。
2. 优先抽出搜索树节点、边、当前路径和回退箭头公共组件，避免 10 支视频各自造轮子。
3. 每支先渲染 `480p15` 做字幕、边界和节奏验收，再生产 `1080p60`。
4. 预览图优先选择“当前路径高亮 + 某个分支回退或被剪掉”的教学关键帧。
5. 成片接入课时后，将章节状态从 `building` 改为 `ready`。
