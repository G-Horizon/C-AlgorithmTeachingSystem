# C05-06 子集枚举

## 课时定位

- 所属章节：第五章 搜索与回溯。
- 核心目标：把子集理解为每个元素独立做一次“不选/选”决策。
- 建议时长：31 秒。
- 目标输出：`SearchSubsetVisualization.mp4`。

## 旁白脚本

1. “一个集合有哪些子集？可以把每个元素都当成一个独立开关。”
2. “轮到元素一时，先走不选它的分支，再走选它的分支。两条边覆盖了全部可能。”
3. “每一层都是同样的二选一。n 个元素就有 n 层决策，叶子数是二的 n 次方。”
4. “所有开关都关闭时，path 是空的，这就是空集。它不能被忘记。”
5. “不选时 path 不变；选择时 push，进入下一层后再 pop。这就是子集的 DFS。”

## 代码画面映射

```cpp
dfs(value + 1);                 // 不选
path.push_back(value);
dfs(value + 1);                 // 选
path.pop_back();
```
