# C05-05 组合枚举

## 课时定位

- 所属章节：第五章 搜索与回溯。
- 核心目标：理解组合不关心选择顺序，用 `start` 让路径严格递增。
- 建议时长：34 秒。
- 目标输出：`SearchCombinationVisualization.mp4`。

## 分镜安排

| 镜头 | 时长 | 画面 | 教学目的 |
| --- | --- | --- | --- |

## 旁白脚本

1. “从一到四中选两个数，组合只关心选了谁，不关心先选谁。”
2. “选了一之后，下一层只能向右看，从二开始。所以路径始终递增。”
3. “这个限制会保留一加二，同时从根源上阻止二加一。同一组数不再因顺序不同重复出现。”
4. “当 path 长度等于 k，当前组合完成。候选从小到大，组合也会按字典序输出。”
5. “如果还缺 need 个数，而右边已经凑不齐，后面的候选可以整段跳过。”

## 代码画面映射

```cpp
int need = k - path.size();
for (int value = start; value <= n - need + 1; value++) {
    path.push_back(value); dfs(value + 1); path.pop_back();
}
```
