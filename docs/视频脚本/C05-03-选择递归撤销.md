# C05-03 选择、递归、撤销选择

## 课时定位

- 所属章节：第五章 搜索与回溯。
- 前置课时：C05-02 DFS 基本框架。
- 核心目标：让学生看清回溯的三步动作以及“恢复现场”的必要性。
- 建议时长：34 秒。
- 目标输出：`SearchChooseUndoVisualization.mp4`。

## 旁白脚本

1. “回溯不是另一种神秘搜索，它是在 DFS 里把可变状态管好。”
2. “先做选择：把候选数放进 path。然后递归，去搜这个选择之下的整棵子树。”
3. “递归返回，说明这棵子树已经搜完。现在要取出刚才的选择，让 path 恢复到进入分支之前。”
4. “只有恢复现场，下一个候选才能站在同样的起点上。如果忘了撤销，兄弟分支就会带着上一条路的垃圾状态。”
5. “记住这个对称结构：选择，递归，撤销。改了几份状态，回来时就恢复几份。”

## 代码画面映射

```cpp
path.push_back(choice);  // 选择
dfs(next);               // 递归
path.pop_back();         // 撤销
```
