# C02-08 快速排序初步

## 课时定位

- 核心问题：一次 partition 怎样把 pivot 放到最终排名位置？
- 学习目标：理解 pivot、扫描边界、分区结果和左右递归区间。
- 成片时长：约 24 秒。

## 分镜安排规划

| 镜头 | 时长 | 画面安排 | 讲解重点 |
| --- | --- | --- | --- |
| 1 | 3s | 标题与“围绕 pivot 分区”出现。 | 先掌握一次分区。 |
| 2 | 4s | 末尾元素标红为 pivot，i、j 指针出现。 | i 维护小区间边界。 |
| 3 | 8s | j 扫描，小值交换进左侧，大值留在右侧。 | 分区循环不变量。 |
| 4 | 4s | pivot 与 `a[i+1]` 交换并标绿。 | pivot 到达最终位置。 |
| 5 | 5s | 左右区间递归、复杂度与优化方向出现。 | 不再包含 pivot。 |

## 视频脚本

| 镜头 | 旁白 | 画面动作 |
| --- | --- | --- |
| 1 | “快速排序的核心，是先围绕一个基准值完成一次分区。” | 标题保留在顶部。 |
| 2 | “这里选最后一个元素作为 pivot。`i` 左侧保存不大于 pivot 的元素，`j` 负责扫描。” | pivot 标红，指针落位。 |
| 3 | “遇到小值，就扩大左侧区域并交换；遇到大值，继续向右扫描。” | 小区间与大区间用不同颜色表示。 |
| 4 | “扫描结束，把 pivot 放到 `i+1`。这个位置就是它排序后的最终位置。” | pivot 交换后变绿。 |
| 5 | “接下来只递归处理 pivot 左右两边。平均复杂度是 `O(n log n)`，极不均衡时会退化。” | 显示递归区间和复杂度。 |
