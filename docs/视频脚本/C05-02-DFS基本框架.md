# C05-02 DFS 基本框架

## 课时定位

- 所属章节：第五章 搜索与回溯。

- 核心目标：掌握“先一条路走到底，再回到最近岔路口”的 DFS 执行顺序。
- 建议时长：32 秒。
- 目标输出：`SearchDfsFrameworkVisualization.mp4`。

## 旁白脚本

1. “深度优先搜索，名字里已经说了它的性格：先往深处走。”
2. “从根选中第一个候选后，DFS 不会立刻回来，而是继续进入下一层，一直到叶子。”
3. “到达出口，输出当前方案，这一层调用结束，执行权回到最近的岔路口。”
4. “现在 for 循环继续下一个候选。就这样，走到底、回退、换路，直到所有叶子都被访问。”
5. “代码只要守住三个位置：递归出口、本层候选、参数向下一层推进。”

## 代码画面映射

```cpp
void dfs(int pos) {
    if (pos == n) { output(path); return; }
    for (int choice = 1; choice <= m; choice++) {
        choose(choice); dfs(pos + 1); undo(choice);
    }
}
```
