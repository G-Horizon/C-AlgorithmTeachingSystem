# C04-07 汉诺塔

## 制作目标

- 学习目标：把复杂搬运任务固定拆成两个同类子任务和一次关键动作。
- 核心结论：移 n-1 个、移最大盘、再移 n-1 个。
- 输出视频：`media/videos/recursion_chapter_scenes/1080p60/RecursionHanoiVisualization.mp4`
- 预览图：`media/previews/recursion_hanoi_preview.png`

## 分镜与旁白

| 镜头 | 画面 | 旁白 |
| --- | --- | --- |
| 1 | 三根柱子与三个盘子出现。 | 目标是把全部盘子从 A 移到 C，并始终保持大盘在小盘下面。 |
| 2 | 第一步卡片：上面 n-1 个从 A 到 B。 | 为了移动最大盘，必须先把它上面的盘子整体移到辅助柱。 |
| 3 | 第二步卡片：最大盘从 A 到 C。 | 最大盘空出来后，这一步可以直接完成。 |
| 4 | 第三步卡片：n-1 个从 B 到 C。 | 最后再解决一个角色不同但结构相同的 n-1 盘问题。 |
| 5 | 三组参数角色交换。 | from、auxiliary、to 在子调用中交换角色，递归骨架本身不变。 |

## 练习衔接

- `recursion-hanoi-moves`：输出标准移动序列。
- `recursion-hanoi-count`：由三步分解得到次数递推。
- `recursion-hanoi-disk-counts`：统计每个编号的盘子被移动几次。
