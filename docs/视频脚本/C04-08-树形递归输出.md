# C04-08 树形递归输出

## 制作目标

- 学习目标：通过访问语句的位置理解前序、中序、后序遍历。
- 核心结论：左右子树是同类子问题，三种遍历只改变访问根节点的时机。
- 输出视频：`media/videos/recursion_chapter_scenes/1080p60/RecursionTreeTraversalVisualization.mp4`
- 预览图：`media/previews/recursion_tree_traversal_preview.png`

## 分镜与旁白

| 镜头 | 画面 | 旁白 |
| --- | --- | --- |
| 1 | 七个节点组成二叉树。 | 一棵树的左右孩子仍然是树，因此可以用同一个函数继续处理。 |
| 2 | 根、左子树、右子树按前序依次变色。 | 把访问放在两次递归之前，就得到根、左、右的前序遍历。 |
| 3 | 底部显示前序序列。 | 动画中节点发光的顺序，就是程序输出的顺序。 |
| 4 | 切换三种访问位置示意。 | 访问放中间是中序，放在两次子调用之后是后序。 |
| 5 | 空节点卡片直接返回。 | 任何树递归都要先处理空节点出口。 |

## 练习衔接

- `recursion-tree-preorder`：根、左、右。
- `recursion-tree-inorder`：左、根、右。
- `recursion-tree-postorder`：左、右、根。
