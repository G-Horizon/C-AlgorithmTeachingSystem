## 旁白脚本

1. “把搜索树放进迷宫，每个格子就是一个状态，上下左右就是候选分支。”
2. “新坐标不能越界，不能撞墙，也不能走进当前路径已经访问过的格子。”
3. “走进一格时立即标记 visited。这样即使通道绕成一个环，DFS 也不会在里面无限打转。”
4. “当四个方向都走不通，这条路失败，递归返回上一个岔路口，再试别的方向。”
5. “只判断连通时，一个格子访问一次就够了。统计所有简单路径时，visited 只属于当前路径，回退时要恢复。”

## 代码画面映射

```cpp
if (outside(nx, ny) || grid[nx][ny] == '#' || visited[nx][ny]) continue;
visited[nx][ny] = true;
dfs(nx, ny);
visited[nx][ny] = false; // 仅枚举所有简单路径时需要
```

## 练习衔接

- `search-maze-reachable`：判断起点与终点是否连通。
- `search-maze-path-count`：统计所有不重复走格子的路径。
