# C05-08 剪枝

## 课时定位

- 所属章节：第五章 搜索与回溯。
- 核心目标：理解剪枝是用可证明的不可能条件提前跳过整棵子树。
- 建议时长：36 秒。
- 目标输出：`SearchPruningVisualization.mp4`。

## 旁白脚本

1. “回溯慢，往往不是每个节点慢，而是节点太多。剪枝要做的，是在走进子树之前证明它没有希望。”
2. “如果所有数都是正数，当前和已经超过目标，后面再怎么选都只会更大。整棵子树可以不走。”
3. “反过来，如果把所有剩余数都加上仍然不到目标，这条路也不可能成功。”
4. “一个很小的 if，跳过的不是一个节点，而是它下面的整棵树。”
5. “但剪枝不能凭感觉。如果剩余数里有负数，当前超标后还可能被拉回。不能证明不漏解，就不能剪。”

## 代码画面映射
