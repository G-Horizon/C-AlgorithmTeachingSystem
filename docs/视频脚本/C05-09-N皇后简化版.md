# C05-09 N 皇后简化版

## 课时定位

- 所属章节：第五章 搜索与回溯。
- 核心目标：按行搜索，用列与两类对角线标记快速检查冲突。
- 建议时长：42 秒。
- 目标输出：`SearchNQueensVisualization.mp4`。

## 旁白脚本

1. “N 皇后要让每个皇后都不能在同一行、同一列和同一条对角线上互相攻击。”
2. “我们按行放，所以每层 DFS 只放一个皇后，行冲突天然消失。放下皇后后，锁定它的列和两条对角线。”
3. “下一行从左到右尝试。只要列、row-col 对角线或 row+col 对角线已被占用，这一格就直接跳过。”
4. “如果整行都没有安全位置，说明前面的选择走进了死路。取下上一个皇后，同时解锁三组标记。”
5. “当 row 等于 n，每一行都已经合法放置。用每行的列号，就能紧凑地记录一个方案。”

## 代码画面映射

```cpp
int down = row - col + n - 1;
int up = row + col;
if (columnUsed[col] || diagonalDown[down] || diagonalUp[up]) continue;
```

- `search-n-queens-first`：用 bool 返回值在第一个方案处提前停止。
