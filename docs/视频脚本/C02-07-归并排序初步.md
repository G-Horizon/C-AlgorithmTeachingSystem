# C02-07 归并排序初步

## 课时定位

- 核心问题：怎样把大数组拆小，再把两个有序段合成一个有序段？
- 学习目标：理解递归拆分、单点出口、双指针合并和稳定性。
- 成片时长：约 27 秒。

## 分镜安排规划

| 镜头 | 时长 | 画面安排 | 讲解重点 |
| --- | --- | --- | --- |
| 1 | 3s | 标题与“先拆再合”出现。 | 建立分治框架。 |
| 2 | 6s | 数组递归拆成长度为 1 的小段。 | 单个元素天然有序。 |
| 3 | 9s | 两个有序段用 i、j 指针写入输出区。 | 每次取较小者。 |
| 4 | 4s | 一侧耗尽，另一侧剩余元素整体接上。 | 合并收尾。 |
| 5 | 5s | 伪代码、`O(n log n)` 与稳定性出现。 | 拆分和合并对应代码。 |

## 视频脚本

| 镜头 | 旁白 | 画面动作 |
| --- | --- | --- |
| 1 | “归并排序做两件事：先把数组不断拆小，再把有序小段合起来。” | 标题保留在顶部。 |
| 2 | “区间从中间分开，直到每段只剩一个元素。单个元素天然有序。” | 递归树逐层展开。 |
| 3 | “合并时，`i` 和 `j` 指向两个有序段当前最小的元素，每次把更小者写入结果。” | 两指针比较，输出格依次变绿。 |
| 4 | “一边用完后，另一边剩余元素可以直接接上。” | 剩余元素连续写入。 |
| 5 | “每层合并总工作量约为 n，共有 log n 层，所以是 `O(n log n)`。” | 高亮递归与 merge。 |
