# C04-05 递归展开与回收：阶乘

## 制作目标

- 学习目标：用阶乘看清“先留下运算，再逐层完成”的递归过程。
- 核心结论：`return n * factorial(n - 1)` 同时描述子问题与合并方式。
- 输出视频：`media/videos/recursion_chapter_scenes/1080p60/RecursionFactorialVisualization.mp4`
- 预览图：`media/previews/recursion_factorial_preview.png`

## 分镜与旁白

| 镜头 | 画面 | 旁白 |
| --- | --- | --- |
| 1 | 左侧从 `5!` 展开到 `5×4×3×2×1`。 | 当前层先留下自己的因子，再请求更小阶乘的答案。 |
| 2 | 最小任务 `1!` 高亮。 | 1 的阶乘可以直接回答，它是乘法链开始回收的位置。 |
| 3 | 右侧从 1、2、6、24 回收到 120。 | 每一层拿到子结果后乘上自己的 n，再返回给父层。 |
| 4 | 连接展开末端与回收起点。 | 展开顺序向下，真正的乘法计算则在返回时向上完成。 |
| 5 | 显示 C++ 返回表达式。 | 一个正确的递归返回式必须说清怎样从子问题答案得到当前答案。 |

## 练习衔接

- `recursion-factorial-basic`：完成标准阶乘递归。
- `recursion-factorial-trace`：打印每层回收后的当前答案。
- `recursion-combination`：把单分支扩展为两个组合数子问题。
