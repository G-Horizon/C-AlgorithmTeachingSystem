## 内容验收标准

1. 章节页可进入全部 8 个课时，练习详情均能由后端题库加载。
2. 示例代码采用 C++17，可直接复制提交；二分区间、重复值、指数 0、全负数和逆序对溢出均有边界覆盖。
3. 章末测验的关联题号必须存在，复习路线不得引用计划中题目。
4. 视频统一使用“蓝色候选、黄色当前、绿色完成、灰色排除”的视觉语义。
5. C07-04/C07-05 可复用第二章排序素材，但必须新增分治递归树与复杂度表达。
