## 产品目标

让学生通过：

```text
看视频 -> 读讲义 -> 理解 C++ 代码 -> 做题提交 -> 获得反馈
```

逐步建立算法直觉，而不是只靠刷题记模板。

## 课程章节

1. 高精度计算。
2. 数据排序。
3. 递推算法。
4. 递归算法。
5. 搜索与回溯算法。
6. 贪心算法。
7. 分治算法。
8. 广度优先搜索算法。
9. 动态规划。
