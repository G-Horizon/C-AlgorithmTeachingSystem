# 第八章“广度优先搜索”内容生产清单

## 当前可交付状态

- 学生课时：9 个，覆盖队列、层序、网格、最短步数、路径还原、多源、骑士、状态 BFS 及 BFS/DFS 对比。
- 练习题：18 道，每课时 2 道，全部接入后端题目详情与判题入口。
- 学习支持：18 道题均有分层提示、解题步骤和常见错误。
- 章节复盘：5 道总结测验、3 组回刷题包。
- 示例代码：所有课时均提供可阅读的 C++ BFS 主循环和对应状态建模。
- 视频：9 支视频已有统一分镜规划，成片与封面待制作，因此课程页暂不声明视频资源。

## 课时与练习矩阵

| 编号 | 课时 | 核心能力 | 配套练习 |
| --- | --- | --- | --- |
| C08-01 | 队列与 FIFO | 正确使用队首、出队、入队 | 队列指令模拟、轮转队列 |
| C08-02 | BFS 层序扩散 | 无权图距离与分层统计 | 单源距离、各层节点数 |
| C08-03 | 网格 BFS | 方向、边界、障碍与访问标记 | 迷宫最短步数、可达格计数 |
| C08-04 | 最短步数为什么成立 | 首次到达最短与隐式图 | 数轴最少操作、模环操作 |
| C08-05 | 路径还原 | 前驱记录、逆推与反转 | 迷宫路径、无权图路径 |
| C08-06 | 多源 BFS | 多起点同时入队与最近源 | 感染时间、最近源距离 |

## 视频续作顺序

1. 先制作 C08-01 与 C08-02，固定队列、当前层、下一层和距离颜色语义。
2. 制作 C08-03 与 C08-04，复用网格组件并突出“入队即标记”。
3. 制作 C08-05 与 C08-06，分别增加前驱箭头和多源波纹。
4. 制作 C08-07 与 C08-08，验证同一 BFS 框架对不同状态的复用。
5. 以 C08-09 收束章节，使用同一迷宫对比 DFS 深入与 BFS 铺层。

## 验收口径

- 每个新状态在入队时完成访问标记，画面与代码同步。
- 队列队首、当前扩展节点、未处理节点和已处理节点有稳定颜色语义。
- 网格动画先判边界，再判障碍与访问状态，不出现越界坐标。
- 多源动画所有源点同时处于第 0 层；路径还原动画明确展示反向收集与最终反转。
- 课程题号在前端与后端完全一致，前端构建、Python 语法和题目 API 校验通过。
